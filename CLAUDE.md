# Zheka Arcade — правила проєкту

## Що це
Аркадний шутер (у стилі Space Invaders) у браузері.

## Стек
- **Phaser 3** — ігровий рушій
- **TypeScript** (strict mode)
- **Vite** — dev-сервер і збірка
- Жодних зовнішніх ассетів: усі текстури генеруються процедурно у `BootScene`.

## Команди
- `npm run dev` — запустити dev-сервер (http://localhost:5173)
- `npm run build` — перевірка типів + продакшн-збірка
- `npm run typecheck` — лише перевірка типів
- `npm run lint` — ESLint

## Структура
- `src/main.ts` — конфіг Phaser і запуск гри
- `src/config.ts` — усі ігрові константи (швидкості, кількість ворогів, тощо)
- `src/scenes/BootScene.ts` — генерація текстур
- `src/scenes/GameScene.ts` — основний геймплей
- `src/scenes/GameOverScene.ts` — екран програшу + рестарт

## Керування
- ← → або A / D — рух
- Space або ↑ — стрільба

## Принципи розробки
1. Малі кроки: одна механіка — один комміт.
2. Усі магічні числа — у `src/config.ts`, не в коді сцен.
3. Перед коммітом: `npm run typecheck` має бути чистим.
4. Кулі переюзовуються через object pool (Phaser group з maxSize) — не створювати нові об'єкти в кожному кадрі.
